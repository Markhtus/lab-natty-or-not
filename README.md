# IA na Vida de um Estudante Universitário

## 📒 Descrição
Este projeto consiste na criação de um mini e-book gerado com Inteligência Artificial mostrando como a IA pode ajudar estudantes universitários em tarefas como estudo, organização e produtividade.

O objetivo foi criar um conteúdo simples, realista e útil utilizando ferramentas de IA generativa.

## 🤖 Tecnologias Utilizadas

- ChatGPT – geração de textos
- Bing Image Creator – geração de imagens
- Canva – montagem do e-book
- GitHub – hospedagem do projeto

## 🧐 Processo de Criação

Primeiro foi definido o tema do projeto: o uso de Inteligência Artificial no dia a dia de estudantes universitários.

Em seguida, foram gerados textos utilizando o ChatGPT com explicações sobre:

- O que é Inteligência Artificial
- Como usar IA para estudar
- Como usar IA para programar
- Como usar IA para organizar tarefas

Depois foram criadas imagens com IA para ilustrar o e-book.

Por fim, o conteúdo foi organizado em formato de e-book utilizando o Canva e publicado no GitHub.

## 🚀 Resultados

O resultado foi um mini e-book simples e informativo mostrando aplicações práticas de Inteligência Artificial para estudantes.

O projeto demonstra o uso de IAs generativas para produção de conteúdo digital.

## 💭 Reflexão

Criar um conteúdo que pareça natural utilizando Inteligência Artificial foi um desafio interessante.

Foi possível perceber que a IA ajuda muito na produtividade, mas ainda é necessário revisar e ajustar os textos para deixá-los mais naturais.
